# Ecommerce-sales-analysis
An interactive **E-commerce Sales Dashboard** built using **Microsoft Power BI** to analyze sales performance, profitability, customer behavior, and regional trends across India.

This project demonstrates strong data analytics, visualization, and business-insight skills, making it suitable for **Data Analyst / Business Analyst fresher roles**.

---

## Project Overview

The dashboard provides a comprehensive view of e-commerce sales data with key KPIs and interactive visuals that help stakeholders:

- Track overall sales and profit
- Analyze performance by state, city, category, and sub-category
- Understand customer purchasing patterns
- Identify profitable and loss-making months
- Compare payment modes and order quantities

## Key KPIs

- **Total Sales Amount:** 438K  
- **Total Profit:** 37K  
- **Total Quantity Sold:** 5,615  
- **Average Order Value (AOV):** 121K  


## Dashboard Features

### Sales Analysis
- Amount by **State**
- Orders by **City** (Map visualization)
- Amount by **Customer Name**

### Profit Analysis
- Profit by **Sub-Category**
- Profit by **Month** (highlighting loss months)

### Customer & Product Insights
- Quantity by **Category** (Clothing, Electronics, Furniture)
- Quantity by **Payment Mode** (COD, UPI, Cards, EMI)

## Tools & Technologies

- **Microsoft Power BI**
- **Power Query** (Data Cleaning & Transformation)
- **DAX** (Calculated Measures & KPIs)
- **Excel / CSV Dataset** (Source data)
- **Data Visualization Best Practices**

## Dataset Details

The dataset includes:
- Order details  
- Customer names  
- State & city  
- Category & sub-category  
- Payment mode  
- Sales amount, profit, and quantity  
- Monthly and quarterly timelines  

## Key Insights

- **Maharashtra** generates the highest sales revenue  
- **Clothing** category contributes the highest quantity sold  
- **Cash on Delivery (COD)** is the most used payment mode  
- Some months show **negative profit**, highlighting cost or pricing issues  
- High-value customers significantly impact total revenue
